# Bar Casa Montjuïc - Analytics Dashboard

Dashboard interactivo para visualizar y comparar métricas de negocio (comensales, ticket medio, transacciones y fidelización) entre el primer trimestre de 2025 y 2026.

## 📊 Características
- **Comparativa Interactiva:** Visualización clara de KPIs (2025 vs 2026).
- **Gráficos Dinámicos:** Implementados con Recharts (Barras y Líneas).
- **Insights de Negocio:** Análisis automático basado en las variaciones de datos.
- **Diseño Responsive:** Optimizado para escritorio y dispositivos móviles con Tailwind CSS.

## 🛠️ Tecnologías
- [React 19](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Recharts](https://recharts.org/)
- [Lucide React](https://lucide.dev/)
- [Motion](https://motion.dev/)

## 🚀 Instalación Local

```bash
npm install
npm run dev
```
